Prompt

Posted at — Feb 1, 2023

Un prompt és una entrada o un conjunt d’entrades que es proporcionen al Model de Machine Learning o IA Generativa per generar una sortida (una imatge, un text, etc.). El prompt pot ser un text, una imatge, una gravació de so o qualsevol altra cosa que es pot utilitzar com a punt de partida per a la generació de nova informació i serveix per a dirigir el model generatiu cap a una direcció específica.
